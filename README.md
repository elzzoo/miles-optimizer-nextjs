# Miles Optimizer - Next.js

Compare cash vs miles for flights with multi-currency support.

## Features

- Flight search with Travelpayouts API
- Airline promotions scraping with SerpAPI
- Multi-currency support (USD, EUR, FCFA)
- Miles value calculation
- Modern UI with Tailwind CSS

## Setup

1. Copy `.env.local.example` to `.env.local`
2. Add your API keys:
   - `NEXT_PUBLIC_TRAVELPAYOUTS_TOKEN`
   - `NEXT_PUBLIC_SERPAPI_KEY`
3. Install dependencies: `npm install`
4. Run development server: `npm run dev`

## Deployment

Deployed on Vercel with automatic deployments from main branch.
